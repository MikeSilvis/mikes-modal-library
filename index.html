Mikes Modal Library

Mikes Modal

Mikes Modal Library is a javascript library to work and behave much like Facebook's Photo viewing experience. Clicking an individual photo will bring up the photo gallery window. It is designed to be used on social networking sites where a photo is on the left and some content is meant for the right side.
